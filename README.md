# Mining Human CAD Joint and Part-Interaction Patterns for AI-Assisted 3D Engineering Workflows

This project studies human CAD assembly and joint patterns using the Fusion 360 Gallery Dataset. The goal is to identify recurring assembly structures, joint behaviors, and complexity profiles that can support future AI-assisted 3D engineering workflows.

## Project Goals
- Understand the structure of assembly and joint data
- Engineer interpretable assembly-level features
- Discover recurring patterns through exploratory data mining
- Cluster assemblies into meaningful structural families
- Visualize complexity, connectivity, and repeated motifs
- Identify promising opportunities for AI-assisted CAD support

## Dataset
This project uses the public Fusion 360 Gallery Dataset, starting with the Assembly-Joint subset.

## Project Structure
- `notebooks/` for step-by-step exploration and analysis
- `src/` for reusable code
- `outputs/` for figures and tables
